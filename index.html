SGCM — Sistema de Gestión de Citas Médicas

Plataforma web hospitalaria para automatizar la gestión de citas

Hospital Regional Traumatológico y Quirúrgico Prof. Juan Bosch (HTQPJB) · La Vega, República Dominicana

Plataforma web hospitalaria desarrollada para automatizar la gestión de citas del Hospital Regional Traumatológico y Quirúrgico Prof. Juan Bosch en La Vega, RD. Construida con FastAPI y PostgreSQL sobre Docker, con autenticación JWT, control de acceso basado en roles en dos capas, auditoría inmutable, generación de reportes PDF con trazabilidad por usuario, y respaldos con verificación SHA-256.

  • 17 casos de uso
  • 3 roles RBAC
  • 197 tests Pytest

  • CU-16 Respaldos local/externo · SHA-256
  • CU-17 Especialidades · catálogo administrable
  • Agenda extendida · PDF + Excel + trazabilidad usuario
